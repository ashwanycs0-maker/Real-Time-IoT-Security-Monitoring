# Real-Time IoT Security Monitoring & Threat Intelligence System

## Overview

This project is a Real-Time IoT Security Monitoring & Threat Intelligence System developed to detect, analyze, and respond to cyber threats in connected environments.

## Key Features

* Real-time device communication using MQTT
* Intelligent threat detection and risk classification
* Live security monitoring dashboard
* Device health and status tracking
* Automated PDF and Excel report generation
* Historical threat analysis and monitoring

## Technologies Used

* IoT
* MQTT
* Python
* Cybersecurity Analytics
* Dashboard Visualization

## Applications

* Smart Homes
* Industrial IoT
* Healthcare Systems
* Smart Cities
* Enterprise Networks
